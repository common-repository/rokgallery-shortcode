Enables you to embed your RokGallery galleries anywhere with a shortcode.

== Description ==

With this plugin you can easily embed your image galleries from RokGallery plugin anywhere with a shortcode. Also adds a meta box to RokGallery Pages screen to dynamically generate the shortcode. Requires the <a href="http://www.rockettheme.com/">RokGallery</a> plugin for WordPress.


== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!
